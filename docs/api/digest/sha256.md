# `Sha256`

[`State`](./state.md) for SHA-256.

## Constructor

### `Sha256()`

Creates a SHA-256 digest state handle.

**Returns:** [`Sha256`](./sha256.md)

```
let state = Sha256()
state.update "abc"
assert_eq $state.digest().hex()
  ba7816bf8f01cfea414140de5dae2223b00361a396177a9cb410ff61f20015ad
```
